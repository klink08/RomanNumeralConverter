In order to run the tests:
    1.) Open command prompt (or your prefered shell)
    2.) Navigate to the root project direct
    3.) Run command "gulp test"

To convert from Arabic to Roman Numeral:
    1.) Open command prompt (or your prefered shell)
    2.) Navigate to the root project direct
    3.) Run command "npm start toRoman *" where * is your desired arabic number to convert

To convert from Arabic to Roman Numeral:
    1.) Open command prompt (or your prefered shell)
    2.) Navigate to the root project direct
    3.) Run command "npm start toArabic *" where * is your desired roman numeral to convert
